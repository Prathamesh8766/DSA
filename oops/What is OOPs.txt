A> What is OOPs?
Ans: OOPs is a practice to organise your code to model real-world things. It allows us to bundle related functions and data into 'objects'. This increases code reusability and helps in structuring a program by bundling associated data and methods into one object.

What is an inheritance with an example?

Ans: Inheritance allows a child class to inherit attributes and methods of a parent class. It promotes code reuse. For example, a Dog and a Cat class are child classes of an Animal class, which inherit the name attribute and speak method from it.

class Animal():
	def __init__(self, name):
		self.name = name
	
	def speak(self):
		raise NotImplementedError("Subclass must implement this method")

class Dog(Animal):
	def speak(self):
		return f"{self.name} says Woof!"

class Cat(Animal):
	def speak(self):
		return f"{self.name} says Meow!"

Inheritance Types:

 1) Single inheritance: There is only one parent and child class, where the child inherits traits of the parent class. A -> B.

 2) Multiple inheritance: Child class inherits traits from multiple parent classes. A, B -> C

 3) Multilevel inheritance: A class inherits from another class, which is a child class of another class. A -> B-> C.

 4) Hierarchical Inheritance: This is when more than one child class inherits from a single parent class. A -> B, C

 5) Hybrid inheritance: A combination of different types of inheritance, usually multiple and multilevel inheritance.



D> What is a Polymorphism with an example?

Ans: Polymorphism means that objects from different classes are treated in the same way. A single function or method can work on different types of objects. For example, the speak() method on both a Dog and a Cat can be called from the same function, but behave differently for each one.

my_dog = Dog("Buddy")
my_cat = Cat("Mona")

def make_animal_speak(animal):
	print(animal.speak())

make_animal_speak(my_dog)
make_animal_speak(my_cat)

Polymorphism Types:
 
 1) Method Overriding (Runtime Polymorphism): A child class provides implementation to the method that is already defined in the parent class. The decision on which method run is made at runtime.

class Animal:
    def speak(self):
        return "Some generic animal sound"

class Dog(Animal):
    def speak(self):
	return "Woof!"

class Cat(Animal):
    def speak(self):
        return "Meow!"

 2) Method Overloading (Compile-time Polymorphism): There are multiple methods with the same name in the same class but with different numbers or types of parameters. The decision on which method run is made at compile-time.

class Calculator:
    def add(self, a, b, c=0):
        return a + b + c

calc = Calculator()
print(calc.add(2, 3))   
print(calc.add(2, 3, 5))

F> What is an abstraction with an example?

Ans: An Abstraction means hiding the complex implementation details and showing only the essential features of an object. For example, a Vehicle class has an abstract method of drive. Any class that inherits Vehicle, like Car, must implement its own derive method.

from abc import ABC, abstractmethod

class Vehicle(ABC):
    @abstractmethod
    def drive(self):
        pass

class Car(Vehicle):
    def drive(self):
        print("The car is driving on the road.")
